# BookRentalSystem


# Instalation & Open project
วิธีการติดตั้งทำได้ดังนี้
1. ดาวน์โหลด source code
2. เข้าไปในโฟลเดอร์ client และ server
3. เปิด cmd
4. ใช้คำสั่งนี้
```bash
npm i
```
5. เปิดโปรเจคด้วยคำสั่งนี้ใน cmd ของทั้ง client และ server
```bash
npm start
```
